# PhiloQuiz

## Qu’est-ce que PhiloQuiz ?

PhiloQuiz est un quiz de révision en philosophie permettant de s’entraîner sur les notions, auteurs et idées du programme de Terminale Génale de philosophie.

---

## Chapitres

* Bonheur
* Devoir
* État
* Justice
* Vérité
* Science & Raison

---

## Modes de révision

* Quiz classique : Une question, 4 réponses dont une seule vraie
  
* Association : Des termes d'un côté, des définitions de l'autre, à vous de les associez
  
* Révision avec trois difficultés disponible :
  - Facile : Vous avez une définition et vous devez retrouvé le mot correspondant
  - Moyen : Majorité de *Définition → Mot* et quelques Mot → Définition
  - Difficile : Vous avez un mot, et vous devez écrire la définition correspondante

* Flashcards : Mots & Définitions
